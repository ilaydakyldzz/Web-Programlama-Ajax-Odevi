# Web Programlama Laboratuvarı - Föy 2 (AJAX)

[cite_start]Bu proje, **Ondokuz Mayıs Üniversitesi Bilgisayar Mühendisliği** Bölümü Web Programlama dersi kapsamında hazırlanmıştır[cite: 302, 303, 306].

## Proje Amacı
[cite_start]AJAX (Asynchronous JavaScript and XML) tekniği kullanılarak, sayfa yenilenmeden sunucu taraflı verilerin (XML) asenkron olarak çekilmesi ve kullanıcı arayüzünün dinamik olarak güncellenmesi hedeflenmiştir[cite: 9, 21].

## Özellikler
* [cite_start]**XMLHttpRequest** nesnesi ile veri transferi[cite: 94, 191].
* [cite_start]XML tabanlı veri saklama yapısı[cite: 59, 262].
* Modern ve etkileşimli CSS tasarımı (Hover ve Active efektleri).
* [cite_start]Hata ayıklama ve durum kontrolü (readyState ve status kontrolü)[cite: 195, 249].

## Kurulum
[cite_start]Dosyaları yerel bilgisayarınıza indirdikten sonra, tarayıcı güvenlik politikaları nedeniyle bir **Local Server** (örneğin VS Code Live Server) üzerinden çalıştırmanız önerilir[cite: 235, 236].
